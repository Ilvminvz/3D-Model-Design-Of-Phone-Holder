## Phone Holder – FreeCAD 3D Model

A simple and stable 3D-printable phone holder designed using FreeCAD.

This project was created as a practical desktop stand for holding smartphones during:
- Charging
- Video calls
- Media viewing
- Desk/workbench usage

The design is lightweight, easy to print, and easy to modify.

---

## Features

- Parametric CAD design
- Designed in FreeCAD
- Compact and lightweight
- Stable structure
- Easy to customize
- 3D-printable
- Multiple export formats included

---

## Files Included

| File Type | Description |
|---|---|
| `.FCStd` | Editable FreeCAD source file |
| `.STL` | Universal 3D printable mesh |
| `.3MF` | Modern 3D printing project format |
| `.STEP` | Engineering CAD exchange format |
